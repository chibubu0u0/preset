# Notion 欄位建立指南

## 必要欄位

| 欄位名稱 | 類型 |
|---|---|
| Original Image | Files & media |
| Edited Image | Files & media |
| AI Status | Status 或 Select |

## AI Status 選項

請建立：

- 未開始
- 分析中
- 已完成
- 失敗

## 建議欄位

| 欄位名稱 | 類型 | 說明 |
|---|---|---|
| AI Style Cluster | Select | AI 歸納出的風格群組 |
| Scene Auto | Select | AI 判斷場景 |
| Lighting Auto | Select | AI 判斷光線 |
| Subject Auto | Select | AI 判斷主體 |
| Color Change Tags | Multi-select | 色彩變化標籤 |
| AI Analysis Summary | Text / Rich text | 繁中分析摘要 |
| Confidence Score | Number | 0–100 信心分數 |
| Training Ready | Checkbox | 是否適合進資料集 |
| AI Error | Text / Rich text | 失敗原因 |

## 使用建議

第一階段你只需要做兩件事：

1. 每列上傳一組 Original Image + Edited Image
2. AI Status 維持「未開始」

分類、場景、光線、色彩標籤都先交給 AI。
